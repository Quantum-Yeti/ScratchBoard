# Scratch Board

**Scratch Board** is a lightweight desktop productivity app that combines note-taking, sticky notes, and fun interactive tools like a built-in Pac-Man game. Designed for quick note capture and playful exploration, it keeps your ideas organized while adding a touch of fun.

---

## Features

- **Sticky Notes:** Create colorful, draggable, and resizable notes with auto-save.  
- **Text Formatting:** Notes support basic text formatting and color customization.  
- **Dashboard:** Organize all your notes in a central, easy-to-access dashboard.  
- **Sidebar Navigation:** Quickly switch between different views of the app.  
- **Pac-Man Game:** Take a break with a built-in, playable Pac-Man game.  
- **Auto-Save:** Notes are automatically saved to the database to prevent data loss.  
- **Custom Colors:** Change the color of your sticky notes using the color picker.  
- **Rounded UI Elements:** Clean and modern rounded corners for notes and interface components.
